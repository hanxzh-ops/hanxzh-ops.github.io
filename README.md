# Personal Portfolio (GitHub Pages)

Starter structure for a GitHub Pages portfolio using the **Minimal Mistakes** Jekyll remote theme.

## Use it
1. Copy these files into your existing `<username>.github.io` repository (or any repo configured for Pages).
2. Edit `_config.yml` and replace placeholders like `YOUR_NAME`, `YOUR_GITHUB_USERNAME`, `YOUR_EMAIL`.
3. Commit + push to your Pages publishing branch.
4. Visit `https://<username>.github.io`.

## Local preview (optional)
If you want to preview locally:
- Install Ruby + Bundler
- Run:
  - `bundle install`
  - `bundle exec jekyll serve`
